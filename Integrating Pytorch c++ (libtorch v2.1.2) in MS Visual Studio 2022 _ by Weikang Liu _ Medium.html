Requirement:

  • libtorch (v2.1.2)
  • MS Visual Studio 2022 (latest build)

Use pre-built libtorch from ‘ https://pytorch.org/ ’ (gpu[cuda] or cpu[none])

  1. Extract the downloaded zip file.
  2. In Visual Studio, access the project properties (right-click on the project icon in Solution Explorer and select “Properties” at the bottom, or press Alt+Enter while the project icon is focused).
  3. Under VC++ Directories > External Include Directories, add $(YOUR_PATH_TO)libtorch\include and $(YOUR_PATH_TO)libtorch\include\torch\csrc\api\include
  4. In Linker > Additional Library Directories, include `$(YOUR_PATH_TO)libtorch\lib`
  5. In C/C++ > Language settings, change Conformance mode to No.
  6. In C/C++ > Language settings, set the C++ Language Standard to “ISO C++17 Standard (/std:c++17)” or higher (C++14 will not work).
  7. In Linker > Input > Additional Dependencies, add all the library files from libtorch\lib: asmjit.lib;c10.lib;c10_cuda.lib;caffe2_nvrtc.lib;cpuinfo.lib;dnnl.lib;fbgemm.lib;fbjni.lib;fmt.lib;kineto.lib;libprotobuf.lib;libprotoc.lib;pthreadpool.lib;pytorch_jni.lib;torch.lib;torch_cpu.lib;torch_cuda.lib;XNNPACK.lib;
  8. Copy and paste all the “.dll” files to the same location as your executable (as shown below).
